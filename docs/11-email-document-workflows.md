# E-Mail- und Dokumenten-Workflows

## 1. E-Mail-Events

### Anfrage eingelangt

Empfänger:
- Antragsteller
- Verwaltung optional

Inhalt:
- Zusammenfassung der Anfrage
- Hinweis, dass noch keine Bestätigung vorliegt

### Buchung bestätigt

Empfänger:
- Antragsteller

Inhalt:
- Zeitraum
- Ressource
- Bedingungen
- Ansprechpartner
- optional PDF-Bestätigung

### Buchung abgelehnt

Empfänger:
- Antragsteller

Inhalt:
- Zeitraum
- Ressource
- Ablehnungsgrund optional

### Erinnerung Abholung

Für Kühlwagen:
- 24 Stunden vor Abholung

### Erinnerung Rückgabe

Für Kühlwagen:
- 24 Stunden vor Rückgabe

## 2. Dokumente

### Buchungsbestätigung PDF

Enthält:
- Buchungsnummer
- Antragsteller
- Ressource
- Zeitraum
- Status
- Hinweise

### Übergabeprotokoll PDF

Enthält:
- Buchung
- Übergabezeit
- Zustand
- Zubehör
- Notizen
- Unterschriftsfelder optional

### Rückgabeprotokoll PDF

Enthält:
- Rückgabezeit
- Zustand
- Reinigung
- Schäden
- Notizen
- Zusatzkosten optional

## 3. Export

CSV/Excel für Admin:

- Zeitraum
- Ressource
- Status
- Antragsteller
- Organisation
- Gebühren
- Kategorie
